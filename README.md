# Make The Pix

## Installation

Cloning repo

```cli
git clone https://github.com/mKsDEV08/makethepix.git
```

Creating Python virtual enviroment

```cli
python3 -m venv .venv
source .venv/bin/activate
```

### alert-server

Installing `alert-server` dependencies

```cli
cd alert-server
pip install -r requirements.txt
```

Starting `alert-server`

```cli
python wsig.py
```
